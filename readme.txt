A simple but robust Widgetized Heads up Display of any given YouTube Channel.

== Description ==

Makes use of Youtube API to show off the popularity stats of any YouTube channel including number of subscribers and total video view count of all videos on the channel, directly from your website!
Using widgets, you can position this in the sidebar, header, or anywhere else you think looks sexy. You can even include the widget multiple times with different account names to show stats for however many channels you own or want to track!  Visit http://ryosa.com for examples.

Features & Uses in v1.0.0:

* Display your Youtube Channel statistics including Total Video Viewcount & Subscribers anywhere on your website
* Track statistics of any other YouTube channel
* Your choice of Compact or Full View (Full enables a header including the channel title)
* Channel Icon can be turned off and on
* Smart links directly to the channel

More configurability coming soon.

== Installation ==

1. Upload zip file contents to the `/wp-content/plugins/` directory or use the wordpress plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new "YES! YouTube Essential Statistics" plugin will be listed, and the settings are all configurable in Appearance > Widgets.
4. In Widgets, drag the YouTube Essential Widget into an expanded slot. 

== Frequently Asked Questions ==

= How do I use the plugin in my theme or in a post/page? =

Modify your theme to include the capability for dynamic widget placement, or use a plugin like "Widgets in Pages" to place it anywhere inside the content area of a post or page.
